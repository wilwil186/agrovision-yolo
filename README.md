# AgroVisión — Visión por Computadora con YOLO

Solución de IA para detección de condiciones visibles en hojas de uva usando YOLOv8, con generación de alertas operativas para priorizar revisiones técnicas en campo.

Proyecto desarrollado como prueba técnica para **AgroVisión**, aplicando visión por computadora a problemas reales de calidad, trazabilidad y priorización en agricultura.
